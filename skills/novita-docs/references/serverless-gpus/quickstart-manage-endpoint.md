## [​](https://novita.ai/docs/guides/serverless-gpus-quickstart-manage-endpoint\#check-serverless-endpoint)  Check Serverless Endpoint

Go to the [Serverless Console](https://novita.ai/gpu-instance/console/serverless) to check all the Serverless Endpoints you have created.Click the **“Dropdown”** button at the bottom right corner of the target serverless endpoint to view the number of running Workers, Worker logs, and advanced operation options for the serverless endpoint.

## [​](https://novita.ai/docs/guides/serverless-gpus-quickstart-manage-endpoint\#modify-serverless-endpoint)  Modify Serverless Endpoint

Click the **“Modify”** button to modify the configuration of the target serverless endpoint in the subsequent pop-up window. Except for the “App Name”, all other configurations can be modified. For details, refer to: [Configure Serverless Endpoint](https://novita.ai/docs/guides/serverless-gpus-quickstart-create-endpoint#step-3-configure-serverless-endpoint).
