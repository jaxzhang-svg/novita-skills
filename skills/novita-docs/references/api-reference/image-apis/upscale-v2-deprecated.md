**The Upscale V2 API is deprecated and will be removed in the future. Please migrate to [Upscale V3](https://novita.ai/docs/api-reference/model-apis-upscale).**

## [​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated\#post-upscale-v2)  POST Upscale V2

An indispensable tool for improving the quality of images generated by Stable Diffusion.

## [​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated\#request-headers)  Request Headers

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-authorization)

Authorization

string

required

## [​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated\#request-body)  Request Body

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-image)

image

string

required

Base64 encoded image

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaler-1)

upscaler\_1

string\|null

AI upscalers are models trained with massive amounts of data.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-resize-mode)

resize\_mode

integer\|null

Resizing mode:

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaling-resize)

upscaling\_resize

number\|null

The magnification factor of the image. For example, if you input upscaling\_resize:2 with an image size of 512x512, it will return images of 1024x1024 (the maximum image size is 2048).

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaling-resize-w)

upscaling\_resize\_w

integer\|null

The target image width; the maximum image size is 2048.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaling-resize-h)

upscaling\_resize\_h

integer\|null

The target image height; the maximum image size is 2048.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaling-crop)

upscaling\_crop

boolean\|null

Zoom in after cropping.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-upscaler-2)

upscaler\_2

string\|null

AI upscalers are models trained with massive amounts of data.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-extras-upscaler-2-visibility)

extras\_upscaler\_2\_visibility

number\|null

Range \[0~1\], Extras upscaler 2 visibility.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-gfpgan-visibility)

gfpgan\_visibility

number\|null

Range \[0~1\], GFPGAN visibility.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-codeformer-visibility)

codeformer\_visibility

number\|null

Range \[0~1\], CodeFormer visibility.

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-codeformer-weight)

codeformer\_weight

number\|null

Range \[0~1\], CodeFormer weight (0 = maximum effect, 1 = minimum effect).

[​](https://novita.ai/docs/api-reference/model-apis-upscale-v2-deprecated#param-img-expire-ttl)

img\_expire\_ttl

integer

Image storage time (seconds). Range \[0, 604800\].
