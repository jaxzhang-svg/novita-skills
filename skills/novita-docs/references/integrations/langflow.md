The integration of Langflow and Novita AI creates a powerful development environment that streamlines the creation of multi-agent and RAG applications. This combination leverages Langflow’s intuitive visual flow builder for rapid prototyping while enabling seamless workflow design and testing capabilities. Novita AI enhances this ecosystem with its advanced Model APIs, featuring industry-leading language models including Llama, DeepSeek, and Mistral.This step-by-step guide will help you access Novita AI LLM API on Langflow and start building AI-powered applications.

## [​](https://novita.ai/docs/guides/langflow\#accessing-novita-ai-llm-api-on-langflow)  **Accessing Novita AI LLM API on Langflow**

### [​](https://novita.ai/docs/guides/langflow\#step-1-expand-the-%E2%80%9Cmodels%E2%80%9D-tag)  Step 1: **Expand the “Models” Tag**

- Find and expand the “Models” section on Langflow’s sidebar.








![Expand the “Models” Tag on Langflow](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/Expandthe%E2%80%9CModels%E2%80%9DTagonLangflow.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=f35e836048716c844e055e5c62ecfb8d)


### [​](https://novita.ai/docs/guides/langflow\#step-2-drag-novita-ai-to-your-canvas)  **Step 2: Drag Novita AI to Your Canvas**

- Select and position the “Novita AI “component from the sidebar onto the Langflow canvas area.








![Drag Novita AI to Your Canvas on Langflow](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/DragNovitaAItoYourCanvasonLangflow.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=b0969297ea155db72d276eef78f53457)


### [​](https://novita.ai/docs/guides/langflow\#step-3-create-a-global-variable-for-the-novita-ai-api-key)  **Step 3: Create a Global Variable for the Novita AI API Key**

- Set up Langflow authentication instantly by adding your Novita AI API Key as a global variable.








![Create a Global Variable for the Novita AI API Key on Langflow](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/CreateaGlobalVariablefortheNovitaAIAPIKeyonLangflow.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=fe7dc4ea3c6389a3eecef9b55b2392a9)


### [​](https://novita.ai/docs/guides/langflow\#step-4-get-your-novita-ai-api-key)  Step 4: **Get your Novita AI API Key**

- Generate Your API Keys Instantly on the [Key Management page](https://novita.ai/settings/key-management).

### [​](https://novita.ai/docs/guides/langflow\#step-5-select-a-model)  Step 5: **Select a Model**

- Choose from Novita AI’s powerful language models: DeepSeek, Llama, and Mistral and then select the perfect AI model for your project needs.








![Select a Model on Langflow](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/SelectaModelonLangflow.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=07c02a78cf3eb65dfe93bd7bfe6e750e)


### [​](https://novita.ai/docs/guides/langflow\#step-6-start-building-your-ai-app)  **Step 6: Start Building Your AI App**

- Create advanced AI functionality by connecting Novita AI components to your Langflow workflow system.
