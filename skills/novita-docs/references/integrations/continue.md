The powerful integration of Continue and Novita AI brings AI-powered development directly into your coding environment. This can help you access intelligent code suggestions, automated testing, and real-time pair programming without switching contexts. Novita AI excels at deploying AI models instantly through our intuitive API, scales effortlessly with our cost-effective GPU infrastructure, and maintains peak performance throughout your development process.Our step-by-step guide walks you through Novita AI implementation on Continue, covering methods of using Continue.dev and integrating Novita AI with Continue using an API Key.

## [​](https://novita.ai/docs/guides/continue\#how-to-use-continue-dev)  How to use Continue.dev?

### [​](https://novita.ai/docs/guides/continue\#vs-code)  VS Code

Step 1: Click `Install`to add the Continue extension from Visual Studio Marketplace.Step 2: Click`Install`again in VS Code to complete the Continue extension setup.Step 3: Move the Continue logo to your right sidebar for enhanced workflow access.

![How to use Continue.dev-VS Code](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/HowtouseContinue.dev-VSCode.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=928c0381da05ae91fb980bd810fb6984)

### [​](https://novita.ai/docs/guides/continue\#jetbrains)  JetBrains

Step 1: Access IDE settings by pressing Ctrl + Alt + S in your JetBrains environment.Step 2: Navigate to plugins and search Continue in the JetBrains marketplace.Step 3: Click `Install`and find the Continue logo on your right toolbar.

![How to use Continue.dev-JetBrains](https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/third-party/HowtouseContinue.dev-JetBrains.webp?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=401839f1cb661b3e5740141ef3bdbead)
