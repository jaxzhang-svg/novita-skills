Novita Agent Sandbox CLI is a command line tool that allows you to list, kill running [sandboxes](https://novita.ai/docs/guides/sandbox-lifecycle), and manage [sandbox templates](https://novita.ai/docs/guides/sandbox-template).

## [​](https://novita.ai/docs/guides/sandbox-cli\#installation)  Installation

You can install the CLI globally using npm:

Bash

Copy

```
npm i -g novita-sandbox-cli
```

Additionally, some new advanced features will be first released in the **Beta versions** of CLI tool, which you can install using the following methods. If you encounter any issues during use of these Beta features, please feel free to [contact us](https://meetings-na2.hubspot.com/junyu).

## [​](https://novita.ai/docs/guides/sandbox-cli\#install-beta-cli)  Install Beta CLI

Bash

Copy

```
npm i -g novita-sandbox-cli@beta
```
