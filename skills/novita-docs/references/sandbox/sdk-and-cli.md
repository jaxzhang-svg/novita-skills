Novita Agent Sandbox provides official stable SDKs (currently supporting Python and JavaScript & TypeScript) as well as accompanying command-line tools to help you easily integrate sandbox capabilities into your business. Below are the installation methods for these components.

## [​](https://novita.ai/docs/guides/sandbox-sdk-and-cli\#install-stable-sdk)  Install Stable SDK

JavaScript & TypeScript

Python

Copy

```
npm i novita-sandbox
```

## [​](https://novita.ai/docs/guides/sandbox-sdk-and-cli\#install-stable-cli)  Install Stable CLI

Bash

Copy

```
npm i -g novita-sandbox-cli
```

Additionally, some new advanced features will be first released in the **Beta** versions of the SDK and CLI tools, which you can install using the following methods. After thorough validation, the Beta versions of the SDK and CLI tools will be regularly synchronized and updated to the stable version. If you encounter any issues during use, please feel free to [contact us](https://meetings-na2.hubspot.com/junyu).

## [​](https://novita.ai/docs/guides/sandbox-sdk-and-cli\#install-beta-sdk)  Install Beta SDK

JavaScript & TypeScript

Python

Copy

```
npm i novita-sandbox@beta
```

## [​](https://novita.ai/docs/guides/sandbox-sdk-and-cli\#install-beta-cli)  Install Beta CLI

Bash

Copy

```
npm i -g novita-sandbox-cli@beta
```
