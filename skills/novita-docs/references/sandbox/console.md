Agent Sandbox console helps you view and manage sandboxes, templates, and usage statistics.

## [​](https://novita.ai/docs/guides/sandbox-console\#sandbox-management)  Sandbox Management

- Display the list of currently running sandboxes.
- Support searching for a sandbox by ID.
- Support filtering sandboxes by creation time, template, vCPUs, memory, etc.

![Sandbox Management](https://mintcdn.com/novitaai/H3Kjvdvlhgt0Aohj/guides/images/sandbox-console-sbx.png?fit=max&auto=format&n=H3Kjvdvlhgt0Aohj&q=85&s=666d0d76871e84223e6ad103d0ea0a7a)

## [​](https://novita.ai/docs/guides/sandbox-console\#template-management)  Template Management

- Display the list of available templates, including official templates provided by Novita official and your [custom templates](https://novita.ai/docs/guides/sandbox-template).
- Support searching for a template by ID.
- Support filtering templates by visibility (public, private, or all), vCPUs, and memory.

![Sandbox Management](https://mintcdn.com/novitaai/H3Kjvdvlhgt0Aohj/guides/images/sandbox-console-template.png?fit=max&auto=format&n=H3Kjvdvlhgt0Aohj&q=85&s=871b288b80b453634576fc1d5d395c96)

## [​](https://novita.ai/docs/guides/sandbox-console\#usage-statistics)  Usage Statistics

Console provides daily resource usage and cost analysis to help you monitor your consumption and optimize your usage.

- **vCPU Hours**  - Display the total vCPU usage time for all sandboxes on the current day (unit: hours).
  - This data can be accurate to the second, and can be used to evaluate the current computing resource usage.
- **Memory Hours**  - Display the total memory usage time for all sandboxes on the current day (unit: GB·hours).
  - The system calculates the memory allocation and usage time for each sandbox during runtime.
- **Usage Costs**  - Display the total cost of resources on the current day (unit: USD).
  - Includes the total cost of all vCPUs and memory. The storage space and templates are currently free to use.

- All resources are billed by the second, and displayed in daily granularity.
- The chart shows the trend of the past 30 days, which can be used to monitor the trend of usage changes.
- This data may not include the latest usage, which may be slightly different from the bill. The final settlement is based on the [billing details](https://novita.ai/billing/details) page.
