Novita AI **GPUs** provides:

- **[GPU Instance](https://novita.ai/docs/guides/gpu-instance-overview):** A dedicated virtual machine with full control over GPU hardware, ideal for long-running, predictable workloads.
- **[Serverless GPUs](https://novita.ai/docs/guides/serverless-gpus-overview):** A fully managed, on-demand GPU service that dynamically scales and charges based on actual usage, perfect for burstable and sporadic tasks.
