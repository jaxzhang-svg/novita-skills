Edit Instance

[Guides](https://novita.ai/docs/guides/introduction) [API Reference](https://novita.ai/docs/api-reference/basic-authentication) [Changelog](https://novita.ai/docs/changelog/19-01-26)

When you need to modify the “Volume Disk Size”, “Expose HTTP Ports” or “Expose TCP Ports” of a GPU instance, you can use [Novita AI](https://novita.ai/) **Edit Instance** feature. Here’s how to use this feature.

1. Go to the [Instances](https://novita.ai/gpus-console/instances) page and locate the instance you want to modify. Click “More Operations - Edit”.
2. In the pop-up edit window, modify the instance configuration and click the “Save” button.

- You can modify the instance configuration while the instance is running, and the instance will not restart after modification;
- Changes are expected to take effect within 1 minute.

- The “Volume Disk Size” configuration can only be increased, not decreased;
- New “Expose HTTP Ports” and “Expose TCP Ports” cannot conflict with ports that are already in use.
