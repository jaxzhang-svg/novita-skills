Novita AI provides official SDKs to help you easily integrate our services into your applications.

## [​](https://novita.ai/docs/guides/model-apis-sdks\#javascript-sdk)  JavaScript SDK

### [​](https://novita.ai/docs/guides/model-apis-sdks\#installation)  Installation

Install our JavaScript SDK using `npm`:

Copy

```
npm i novita-sdk
```

### [​](https://novita.ai/docs/guides/model-apis-sdks\#quick-start)  Quick Start

Visit [https://github.com/novitalabs/javascript-sdk](https://github.com/novitalabs/javascript-sdk) for a quick start.

## [​](https://novita.ai/docs/guides/model-apis-sdks\#python-sdk)  Python SDK

### [​](https://novita.ai/docs/guides/model-apis-sdks\#installation-2)  Installation

Install our Python SDK using `pip`:

Copy

```
pip install novita-client
```

### [​](https://novita.ai/docs/guides/model-apis-sdks\#quick-start-2)  Quick Start

Visit [https://github.com/novitalabs/python-sdk](https://github.com/novitalabs/python-sdk) for a quick start.
