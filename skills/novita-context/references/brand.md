# Novita Brand Overview

## About Novita AI

**Novita AI** is an AI & Agent Cloud for Developers, providing simple, reliable, and cost-effective cloud infrastructure for running AI models. The platform empowers developers with scalable, production-ready model inference so they can focus on building AI applications.

### Mission

Ship models and agents in minutes, call 200+ models with one API, and run secure, fast agent sandboxes—developer-first and startup-friendly.

## Core Value Proposition

- **Developer-first DX**: Simple APIs/SDKs, clear docs, instant scale
- **Performance**: High throughput LLM serving and low-latency startup for agents
- **Global & reliable**: Deploy close to users on resilient infrastructure
- **Cost-efficient**: Save up to 50% with smart pricing and spot where it fits
- **Bring your own model**: Private endpoints & custom SLAs when you need them
- **Secure by design**: Sandbox isolation for agent workloads

## Product Portfolio

### Model APIs
Run 200+ AI models with a simple API. Plug-and-play access to the latest LLMs, image, video, TTS, embeddings models and more via simple APIs. Scale from prototype to production without managing infrastructure.

### Custom Models
Enterprise-Grade Custom Models with Zero Infrastructure Hassle. Deploy custom models with guaranteed performance SLAs, limitless scalability, and round-the-clock monitoring—no DevOps needed. Focus on innovation, not infrastructure.

### GPU Cloud
Globally distributed GPUs with local speed. Launch high-performance GPU instances in seconds across global regions. Ideal for training, finetuning, and high-throughput inference. Use on-demand or spot at 50% off.

### Agent Sandbox
Safe, instant runtimes for AI agents. Run autonomous agents in isolated containers with ~200 ms startups, safe tool use (browser/API/code), and massive concurrency. Billed per-second for CPU/RAM.
