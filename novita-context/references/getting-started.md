# Getting Started with Novita AI

## Quickstart Guide

Novita AI is the go-to inference platform for AI developers seeking a low-cost, reliable, and simple solution for shipping AI models.

### 1. Log into novita.ai

You can log in using Google or GitHub authentication, and a new account will be automatically created on your first login. You can also sign up using your email address.

**Console**: https://novita.ai/user/login

### 2. Manage API Key

Novita AI authenticates API access using Bearer authentication with an API Key in the request header:
```
Authorization: Bearer {API Key}
```

**Key Management**: https://novita.ai/settings/key-management

### 3. Maintain sufficient credit balance

Novita offers new users a voucher with some credit to try products. To add more credit, visit Billing and Payments and follow the Payment Methods guide.

**Billing**: https://novita.ai/billing

**Recommended**: Setup Automatic Top-up to avoid service interruptions.

## Introduction

Novita provides simple, reliable, and cost-effective cloud infrastructure for running AI models. The platform empowers developers with scalable, production-ready model inference so they can focus on building AI applications.

### Core Products

1. **Model APIs** - Ship top AI models with a simple API
2. **GPUs** - Get dedicated GPUs for running your own models
3. **Agent Sandbox** - Securely run AI-generated code with full isolation and seamless agent integration

### Authentication

All API requests require authentication via Bearer token in the Authorization header:
```
Authorization: Bearer <Your API Key>
```

Get your API key from: https://novita.ai/settings/key-management
